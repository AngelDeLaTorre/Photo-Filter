II- This folder contains the following:\
- tester.py = program in python for guy and executable file\
- nopho.gif = a simple picture(Do not execute with this initial picture)\
- main.cpp = c++ code  from which executable file is created\
- imageproc = executable file that hold both blur algorithms(sequential and parallel implementations)\
- CMakeLists.txt = file needed for cmake software\
\
III- Instruccions:\
1) Open and run python file tester.py\
2)Click on browse button and select a preferred picture\
3)Using the radio buttons select the type of blur(gaussian or radial = motion)\
4) Click the execute button\
5) Click the refresh button to see the filtered image and its respecting graph(x= # of threads, y = time in ms)\
\
P.D. Executable file runs only in unix base machines(No windows)}
